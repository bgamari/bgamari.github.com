Configuration Ubuntu around XMonad

My XMonad configuration can be found on Github. In addition, there are some miscellaneous fixes that need to be done.

Disable global menus

In ~/.gnomerc,

 STARTUP="env UBUNTU_MENUPROXY= $STARTUP"

Install taffybar

 $ cabal install taffybar

Enable dbus support in PulseAudio

Add the following in /etc/pulse/default.pa,

load-module module-dbus-protocol
