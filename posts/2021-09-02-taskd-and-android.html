taskd, foreground and Android

2021-10-02 | tagged with
  • tasks

Android only supports.

Enable NixOS services.taskserver module.

openssl x509 -inform PEM -outform DER -in /var/lib/taskserver/keys/ca.cert -out ca.crt

Copy ca.crt to the device’s internal storage.

On the device, go to Settings -> Biometrics and Security -> Other security settings -> Install from device storage -> CA certificate. Select ca.crt.
